Autoboxing & Unboxing

  • Autoboxing is the automatic conversion that the Java compiler makes between the primitive types and their corresponding object wrapper classes.
    Int -> Integer, double -> Double, ...
  • Converting an object of a wrapper type to its corresponding primitive value is called unboxing.
    Integer -> Int, Double -> double, ...
  • Example:
    ArrayList<Integer> nums = new ArrayList<Integer>();
    nums.add(1); (autoboxing, int priitive to Integer object)
    int b = nums.get(1); (unboxing, Integer object inner value)

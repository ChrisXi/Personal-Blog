Minimum Coin Change Problem

  • Problem: Given a matrix of 0’s and 1’s (binary matrix). Find out Max­i­mum size square sub-matrix with all 1’s.

    111010
    111110
    001110
    111111

    int[][] dp = new int[h][w];
    dp[i][0] = matrix[i][0];
    dp[0][j] = matrix[0][j];

    for (int i=1; i<h; i++)
          for (int j=1; j<w; j++) {
               if (dp[i][j] == 0) dp[i][j] = 0;
               else {
                    int up = dp[i-1][j];
                    int left = dp[i][j-1];
                    if (up == left)

                    else
                         dp[i][j] = Math.min(up, left)+1;

Minimum Coin Change Problem

  • Problem: Given a amount ‘A’ and n coins with different values. Write a pro­gram to find out min­i­mum num­bers of coins required to make the change for the amount ‘A’.
  • Coins set: {1,2,5}, amount: 10
  • 1. Initialization.

    int[][] dp = new int[coins.length+1][amount+1];
    dp[i][0] = -1;
    dp[0][j] = -1;

    if (coins[i-1] > j) dp[i][j] = dp[i-1][j];
    else if (coins[i-1] == j) dp[i][j] = 1;
    else ...

  • 3. Compare current amount with current coin value (smaller). Get the minimum coins number with current amount minus current coin value: dp[i][j-coin[i-1]], the minimum coins number without current coin: dp[i-1][j].

    int no1 = dp[i][j-coin[i-1]];
    int no2 = dp[i-1][j];
    dp[i][j] = (no1==-1 ? no:no+1);
    dp[i][j] = (no2==-1 ? dp[i][j]:min(dp[i][j], no2));

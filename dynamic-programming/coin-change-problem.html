Coin Change

  • Given a set of coins and amount, Write an algoÂ­rithm to find out how many ways we can make the change of the amount using the coins given.
  • Coins set: {1,2,5}, amount: 10
  • 1. Initialization.

    int[][] dp = new int[coins.length+1][amount+1];
    dp[0][j] = 0;
    dp[i][0] = 1;

  • 2. Compare current coin value with current amount (equal or larger). Get the number of ways with current amount minus current coin value: dp[i][j-coin[i-1]], the number of ways without current coin: dp[i-1][j].

    int no1 = dp[i][j-coin[i-1]];
    int no2 = dp[i-1][j];
    dp[i][j] = no1 + no2;

1. File Command

  • C+x C+f : Opens a file, asks for the file name. If it cannot find the file, creates the file
  • C+x C+s : Saves the file without a prompt
  • C+x s : Saves all files with a prompt
  • C+s C+w : Saves the file with a different name. Asks you for the name.
  • M+x "recover-file" : recovers the auto-saved file
Last Updated on: Friday, May 20, 2016

2. Copy/Cut/Paste Command

  • C+k : Kills/deletes the whole lines, puts it into the clipboard
  • C+y : Pastes whatever is in the clipboard at the cursor (subsequent C+y's will keep on pasting)
  • C+s : Starts marking/highligting a region
  • M+w : Copies this region into the clipboard
  • C+w : Cuts this regions into the clipboard (deletes the region and copies it to clipboard)
  • C+/ : Undo
  • C+g C+/ : Redo
Last Updated on: Friday, May 20, 2016

3. Cursor Command

  • fn+-> : Move cursor to the end of the file
  • fn+<- : Move cursor to the head of the file
  • C+a : Beginning of line
  • C+e : End of line
  • M+-> : End of word
  • M+<- : Beginning of word
Last Updated on: Friday, May 20, 2016

4. Buffer Management Command

  • C+x b : Switches buffers, asks you which buffer to switch to.
  • C+x C+b : Switches buffers, but shows you the list of buffers in a new window.
  • C+x k : kill buffer
  • C+x o (other window) : Go to the other window and hit enter on the buffer you want to switch to
  • C+x 0 : Will close that window
  • C+x 1 : Will leave only one window
  • C+x 2 : Will make a horizontal cut and show a secondary window
  • C+x 3 : Will make a vertical cut and show a secondary window
  • C+x C+q : Make the current buffer editable (editable Dired which could be treated as a text buffer)
  • M+x clean-buffer-list : Clean buffer list
  • M+x ido-mode : Switch between buffer
Last Updated on: Friday, May 20, 2016

5. Search Command

  • C-s : Searches forward as you type (When you are within a search, C-s will find the next occurance.)
  • M-C-s : Searches a regexp (Again, hitting C-s while you are in regexp search will find the next occurance.)
  • M-% : Searches and replaces
  • M-s o : Searches and shows alll occurance in an *Occur* buffer (You can click on the lines to jump to those lines)
  • M-x grep : Greps a pattern in the files you specify and shows the results in a *Grep* buffer
  • M-x rgrep: Will recursively grep in a directory (all the files and subdirectories)
Last Updated on: Friday, May 20, 2016

6. Add Self-Designed Command

  • M-x command-name : Will execute the command
  • M-x pwd : Print the working directory
  • M-x indent-region: Indents the region
  • M-x and type anything and hit TAB to auto-complete the commands (or to see the available commands)
  • C-h v : See the value of a veriable
  • C-x C-e : Evaluate a lisp statement at the cursor
Last Updated on: Friday, May 20, 2016

7. Other Command

  • C-x C-c : Quit Emacs
  • C-g : Always quits your command.

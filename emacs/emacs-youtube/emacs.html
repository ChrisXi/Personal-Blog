1. Getting Started and Playing Games

(1) Ctrl+h i : bring up info tabel
(2) Ctrl+g : give up current command
(3) u : go upper level of page (dir)
(4) l : go last viewed page (dir)
(5) Ctrl+s : search for word (Ctrl+s to next word; press enter to end search)
(6) Ctrl+x k : kill buffer
(7) Ctrl+h m : help on the mode
(8) Ctrl+x o : switch to other window
(9) Ctrl+x 1 : return to single view mode

2. Customizations and Themes

(1) Meta+x customize <- : enter customize menu

-> 'Environment', -> 'Frame', -> 'Menu Bar Mode', -> Toggle (choose on/off), -> State (choose 0, 1, 2, 3, ...)

-> 'Environment', -> 'Frame', -> 'Tool Bar Mode', -> Toggle (choose on/off), -> State (choose 0, 1, 2, 3, ...)

-> 'Environment', -> 'Frame', -> 'Cursor', -> Blink Cursor Blinks -> Toggle (choose on/off), -> State (choose 0, 1, 2, 3, ...)

(2) Ctrl+x Ctrl+f : find file

C+x C+f ~/.emacs

(3) Set font

-> 'Face', -> 'Basic Faces', -> 'Default face', -> 'Cursor', -> Blink Cursor Blinks, -> ..., -> State (choose 0, 1, 2, 3, ...)

(4) Meta+x customize-themes <- : set color themes (search 'theme')
(5) Ctrl+x b "name": get back to the "name" buffer (if name was empty, it will get back to the last buffer)

3. Installing Packages and Extensions

(1) Meta-x list-packages <- : view the list of packages
(2) M-x customize-group <-, package <- : enter install packages origin

-> 'Package Archives', -> 'INS', -> write Archive Name / URL or directory name

(3) Shift-u : mark package for upgrading
(4) i (before the package name) x : install package (5) Choose package:

Choose package (referece 2.4)

Open ~/.emacs file; Add (package-initlize) on the top (enable to load new theme when open emacs)

(5) Install/Enable nyan-mode package.

Meta+x nyan-mode (repeat to disable)

4. Exploring the Filesystem with Dired

(1) Ctrl+x Ctrl+f <- : open the current directory, enter the Dired mode
(2) Ctrl+s <- : quick jump to and open the file/folder
(3) d x : delete a file/folder(could mark multiple file then delete all of them)
(4) + "name" : create a new directory named "name"
(5) Ctrl+x Ctrl+f : create a new file
(6) Ctrl+s : save
(7) g : refresh the directory
(8) ! : run shell command on the file

! (on the line of the file just created), open -a "Google Chrome"

(9) Shift+c "new name" : copy file with new name of "new name"
(10) m : mark the files

mark multiple files and open them with Google Chrome

(11) u : unmark file
(12) Shift+u : unmark all the files
(13) t : mark all the file (repeat to unmark)

In fact, it's like the operator of negation (toggle)

(14) * / : mark all the folders
(15) Shift+a : search for key word
(16) R "new name": rename the file (R should be the upper case)
(17) s : change the file sorting order
(18) Ctrl+x Ctrl+q : make the current buffer editable (editable Dired which could be treated as a text buffer)
(19) ~ : mark all the backup files

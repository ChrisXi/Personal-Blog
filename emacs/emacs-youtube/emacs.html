1. Getting Started and Playing Games

  • Ctrl+h i : bring up info tabel
  • Ctrl+g : give up current command
  • u : go upper level of page (dir)
  • l : go last viewed page (dir)
  • Ctrl+s : search for word (Ctrl+s to next word; press enter to end search)
  • Ctrl+x k : kill buffer
  • Ctrl+h m : help on the mode
  • Ctrl+x o : switch to other window
  • Ctrl+x 1 : return to single view mode

2. Customizations and Themes

  • Meta+x customize <- : enter customize menu

    • -> 'Environment', -> 'Frame', -> 'Menu Bar Mode', -> Toggle (choose on/off), -> State (choose 0, 1, 2, 3, ...)

    -> 'Environment', -> 'Frame', -> 'Tool Bar Mode', -> Toggle (choose on/off), -> State (choose 0, 1, 2, 3, ...)

    -> 'Environment', -> 'Frame', -> 'Cursor', -> Blink Cursor Blinks -> Toggle (choose on/off), -> State (choose 0, 1, 2, 3, ...)

  • Ctrl+x Ctrl+f : find file

    • C+x C+f ~/.emacs

  • Set font

    • -> 'Face', -> 'Basic Faces', -> 'Default face', -> 'Cursor', -> Blink Cursor Blinks, -> ..., -> State (choose 0, 1, 2, 3, ...)

  • Meta+x customize-themes <- : set color themes (search 'theme')
  • Ctrl+x b "name": get back to the "name" buffer (if name was empty, it will get back to the last buffer)

3. Installing Packages and Extensions

  • Meta-x list-packages <- : view the list of packages
  • M-x customize-group <-, package <- : enter install packages origin

    • -> 'Package Archives', -> 'INS', -> write Archive Name / URL or directory name

  • Shift-u : mark package for upgrading
  • i (before the package name) x : install package
  • Choose package:

    • Choose package (referece 2.4)

    Open ~/.emacs file; Add (package-initlize) on the top (enable to load new theme when open emacs)

  • Install/Enable nyan-mode package.

    • Meta+x nyan-mode (repeat to disable)

4. Exploring the Filesystem with Dired

  • Ctrl+x Ctrl+f <- : open the current directory, enter the Dired mode
  • Ctrl+s <- : quick jump to and open the file/folder
  • d x : delete a file/folder(could mark multiple file then delete all of them)
  • + "name" : create a new directory named "name"
  • Ctrl+x Ctrl+f : create a new file
  • Ctrl+s : save
  • g : refresh the directory
  • ! : run shell command on the file

    • ! (on the line of the file just created), open -a "Google Chrome"

  • Shift+c "new name" : copy file with new name of "new name"
  • m : mark the files

    • mark multiple files and open them with Google Chrome

  • u : unmark file
  • Shift+u : unmark all the files
  • t : mark all the file (repeat to unmark)

    • In fact, it's like the operator of negation (toggle)

  • * / : mark all the folders
  • Shift+a : search for key word
  • R "new name": rename the file (R should be the upper case)
  • s : change the file sorting order
  • Ctrl+x Ctrl+q : make the current buffer editable (editable Dired which could be treated as a text buffer)
  • ~ : mark all the backup files

5. Managing and Finding Buffers

  • Ctrl+x Ctrl+b : get buffer list
  • Meta+x clean-buffer-list : clean buffer list
  • Meta+x ido-mode : Switch between buffer

    • Ctrl+s : forward

    Ctrl+r : backward

  • Turn on ido mode search behavior

    • search ido <-, -> 'Ido Mode', -> State (1)

  • Distinguish buffer with same name under different directory

    • Meta-x customize-group, -> 'Uniquify Buffer Name Styles', -> (choose one style)

  • Meta+x occur "content" : find all the lines that match your regular expression
  • Meta+x multi-occur-in-matching-buffers "filename" "content": find all the lines that match your regular expression in all buffers

    • E.g. Meta+x multi-occur-in-matching-buffers .* new

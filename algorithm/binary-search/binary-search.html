Binary Search

  • The basic binary search, find the given target occurance (if there duplicate exists, it will return the first index).
    int binarySearch(int[] nums, int target) {
          int l=0, h=nums.length-1; //initial low, high pointers
          while (l <= h) {
               int m = l+((h-l)>>1); //get medan pointer
               if (target <= nums[m]) h = m-1;
               else l = m+1;
          }
          return l<=nums.length-1&&num[l]==target ? l:-1;
    } 
    Example:
target: 1
|  1  2 |   1   2 |    1  2 |
|  ↑  ↑ |   ↑   ↑ | ↑  ↑    |
| l/m h | l/m/h   | h  l    |

target: 2
|  1  2 | 1   2   | 1  2 |
|  ↑  ↑ |     ↑   | ↑  ↑ |
| l/m h |   l/m/h | h  l |

target: 3
| 1  2  3  3  4 |  1  2  3  3  4 | 1   2  3  3  4 | 1  2  3  3  4 |
| ↑     ↑     ↑ |  ↑  ↑          |     ↑          |    ↑  ↑       |
| l     m     h | l/m h          |   l/h/m        |    h  l       |
    Note: After the end of while loop, the high pointer will be always in the front of the low pointer (l = h+1). We will use this feature to solve the coming up problems.
  • Find the last occurance of given target.
    int binarySearch(int[] nums, int target) {
          int l=0, h=nums.length-1;
          while (l <= h) {
               int m = l+((h-l)>>1);
               if (target+1 <= nums[m]) h = m-1;
               else l = m+1;
          }
          return 0<=h&&num[h]==target ? h:-1;
    }
  • Find the occurance of the item which was smallest among those items that are larger than the given target.
    int binarySearch(int[] nums, int target) {
          int l=0, h=nums.length-1;
          while (l <= h) {
               int m = l+((h-l)>>1);
               if (target+1 <= nums[m]) h = m-1;
               else l = m+1;
          }
          return l<=nums.length-1 ? h:-1;
    }
  • Find the occurance of the item which was largest among those items that are smaller than the given target.
    int binarySearch(int[] nums, int target) {
          int l=0, h=nums.length-1;
          while (l <= h) {
               int m = l+((h-l)>>1);
               if (target <= nums[m]) h = m-1;
               else l = m+1;
          }
          return h;
    }

• Given two string sequences, write an algo­rithm to find the length of longest sub­se­quence present in both of them.
  • String str1 = "acbaed";
    String str2 = "abcadf";
  • 1. Initialization.

    int len1 = str1.length()+1, len2 = str2.length()+1;
    int[][] dp = new int[len1][len2];
    dp[0][j] = 0;
    dp[i][0] = 0;

  • 2. Condition1.

    for (int i=1; i<=len1; i++)
          for (int j=1; j<=len2; j++) {
               char c1 = str.charAt(i-1);
               char c2 = str.charAt(j-1);
               int up = dp[i-1][j];
               int left = dp[i][j-1];
               dp[i][j] = (c1==c2 ? dp[i-1][j-1]+1:Math.max(up, left));
          }
